Line Following Robot

Tools: Arduino, IR sensors, DC motors, motor driver

Problem / Goal:
To design and implement an autonomous mobile robot capable of detecting and following a predefined path using sensor feedback and basic control logic.

Approach / Design:
The system will use infrared sensors to detect contrast between the line and the surface. Sensor readings will be processed by a microcontroller, which will generate control signals to adjust motor speeds through a motor driver. The robot’s movement will be corrected in real time based on sensor input.

Skills & Concepts Used:

  • Embedded systems fundamentals
  • Sensor interfacing and signal interpretation
  • Basic control logic and decision-making
  • Motor control using a microcontroller

Status: Planned (design and learning phase)

Temperature Monitoring System

Tools: Arduino, temperature sensor, LCD display

Problem / Goal:
To design an embedded system capable of measuring environmental temperature and displaying real-time data in a clear and readable format.

Approach / Design:
A temperature sensor will be interfaced with a microcontroller to acquire analog data, which will be converted to digital values and processed. The processed temperature readings will be displayed on an LCD screen, introducing basic real-time system behavior.

Skills & Concepts Used:

  • Analog-to-digital conversion
  • Sensor calibration and data interpretation
  • Embedded system design
  • Real-time data display

Status: Planned (conceptual design stage)
